# Project Roadmap Visualizer

A tool to create, visualize, and manage project milestones and tasks in a timeline or kanban-style board.

## Features
- Add/edit milestones and tasks
- Timeline and kanban board views
- Drag-and-drop task management
- Color-coded statuses (todo, in progress, done)
- Export roadmap as image (requires [html2canvas](https://html2canvas.hertzen.com/))

## Usage
1. Open `index.html` in your browser.
2. Add milestones and tasks.
3. Switch between timeline and kanban views.
4. Drag tasks between columns in kanban view.
5. Export your roadmap as an image.
