# Visual Commit Explorer

A visual tool to explore and understand git commit history. Users can load a repository and see commits as an interactive graph, with details, diffs, and author stats.

## Features
- Visual commit graph (branches, merges, tags)
- Click to view commit details and file diffs
- Author contribution stats and activity heatmap
- Filter/search by author, message, or date
- Responsive, works in dark and light mode

## Usage
1. Open `index.html` in your browser.
2. (Demo only) Interact with the sample commit graph and UI.
3. For real git data, integrate a backend or WASM git parser.

## Tech
- Vanilla HTML, CSS, JavaScript

## Responsive
- Fully responsive for desktop and mobile.
