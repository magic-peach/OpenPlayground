File Commands

SECTION 1/10

cat

file viewer

Concatenate and display file contents

cat filename.txt cat file1.txt file2.txt > combined.txt
-n number lines -E show line endings

cp

file copy

Copy files and directories

cp source.txt destination.txt cp -r dir1/ dir2/
-r recursive -i interactive

mv

file move

Move or rename files

mv oldname.txt newname.txt mv file.txt /target/directory/
-i prompt before overwrite -v verbose

rm

dangerous

Remove files or directories

rm file.txt rm -rf directory/
-r recursive -f force -rf CAUTION!

touch

file create

Create empty file or update timestamp

touch newfile.txt touch -t 202312310000 file.txt

find

search tool

Search for files in directory hierarchy

find . -name "*.txt" find /home -type f -size +1M
-name search by name -type file type

Directory Commands

SECTION 2/10

ls

directory listing

List directory contents

ls -la ls -lh
-l long format -a show all -h human readable

cd

navigation

Change directory

cd /var/log cd .. cd ~

pwd

info

Print working directory

pwd

mkdir

create

Create directories

mkdir new_directory mkdir -p path/to/new/dir
-p create parent dirs

rmdir

remove

Remove empty directories

rmdir empty_dir

tree

visual

Display directory tree

tree tree -L 2
-L depth level

How to Use This Guide

  • Click on categories in the sidebar to navigate
  • Each command card shows usage examples and flags
  • Use the interactive terminal at the bottom to practice

Quick Tips

  • man command - View manual for any command
  • command --help - Quick help for command
  • Press Tab for auto-completion in real terminals
  • Ctrl+C to cancel a running command
